Container Deployment

This demo is fully containerized and can be deployed anywhere Docker is supported:

# Build the container
docker build -f Dockerfile.demo -t runescape-competition-demo .

# Run the demo
docker run -p 5000:5000 runescape-competition-demo

# Access at http://localhost:5000

The containerized approach ensures consistent deployment across different environments and showcases modern DevOps practices.
